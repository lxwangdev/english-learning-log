## 📖 Today's Reading
- **Source:** [Kubernetes Overview](https://docs.docker.com/get-started/docker-overview/)

Docker provides the ability to package and run an application in a loosely isolated environment called a container. The isolation and security let you run many containers simultaneously on a given host. Containers are lightweight and contain everything needed to run the application, so you don't need to rely on what's installed on the host. You can share containers while you work, and be sure that everyone you share with gets the same container that works in the same way.

## 💭 Reflections

第一句：
原句： Docker is a containers about lightweight,loosely,isolated environment.

主谓一致： a containers 错误。a 是单数，containers 是复数。如果是指 Docker 这个技术，建议用 platform 或 technology；如果是指容器本身，用单数 a container。

介词使用： about 在这里不太自然。通常我们会说 Docker provides（提供）环境，或者容器 is（是）一种环境。

形容词/副词： loosely 是副词，后面通常接形容词（如 loosely coupled，松耦合）。如果你想表达环境是“松散的”，通常直接用形容词。

最接近原意	Docker provides lightweight, loosely coupled, and isolated container environments.	语法严谨，保留了你所有的关键词。

定义式	A Docker container is a standardized, isolated software environment.	侧重于解释“容器是什么”。

Docker is a lightweight, loosely isolated container environment.
